Cepat atau lambat _Forntend_ _Developer_ akan menggunakan _Arrays_ dan _Objects_. <br/>

Mendapakan _value_ dari _array_ yang memiliki lebih dari satu element mungkin memerlukan beberapa baris code.
tapi dengan ada nya fitur _Destructuring_ dari _EchmaScript 6_ (ES 2015) maka kita akan mendapatkan hasil yang sama dengan baris code yang lebih sedikit.

Sebelum ada _EchmaScript 6_ (ES 2015) untuk mengekstraksi _value_ dari sebuah _array_, kamu harus melakukan seperti ini:

```js
const fruits = ['apple', 'banana', 'orange'];

const apple = fruits[0];
const banana = fruits[1];
const orange = fruits[2];

console.log('i have a', apple, 'and a', banana, 'and an', orange);

Output: i have a apple and a banana and an orange
```

Kode diatas tentu berjalan dengan baik, tapi kita menggunakan 3 baris kode untuk mendapakan 3 _value_,
ini baru 3 _element_ bayangkan jika kita ingin memperolah 5 _value_ atau lebih tentu baris kode yang digunakan akan banyak juga.

Sekarang mari kita coba menggunakan fitur _Destructuring_.

```js
const fruits = ['apple', 'banana', 'orange'];


// index 0, 1, 2
const [apple, banana, orange] = fruits;

console.log('i have a', apple, 'and a', banana, 'and an', orange);

Output: i have a apple and a banana and an orange
```

Kita mendapatkan hasil yang sama dengan 1 baris kode diatas, kita tidak perlu menuliskan 3 baris kode lagi.
